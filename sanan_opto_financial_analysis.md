# 三安光电财务分析报告

## 执行摘要

**公司名称**: 三安光电股份有限公司 (600703.SH)
**股票代码**: 600703.SH (上海证券交易所)
**分析日期**: 2026年3月11日
**分析周期**: 2023-2025年
**投资评级**: 谨慎乐观
**目标价格**: 待定

### 核心结论
三安光电作为中国LED芯片行业的龙头企业，在传统LED业务保持领先地位的同时，积极向第三代半导体（化合物半导体）领域转型。公司受益于新能源汽车、5G通信、数据中心等新兴市场需求，但面临行业周期性波动和技术迭代风险。Mini/Micro LED和碳化硅业务是未来主要增长点。

## 1. 公司概况

### 1.1 基本信息
- **成立时间**: 2000年11月
- **上市时间**: 2008年7月（上海证券交易所）
- **总部地点**: 厦门市
- **实际控制人**: 林秀成家族
- **员工人数**: 约15,000人（2024年）

### 1.2 主营业务
1. **LED外延片及芯片**: 传统LED芯片，市场份额全球领先
2. **化合物半导体**: 砷化镓、氮化镓、碳化硅等第三代半导体
3. **Mini/Micro LED**: 新型显示技术，应用于高端显示
4. **射频前端**: 5G通信射频器件
5. **光通讯**: 光芯片和光模块

### 1.3 市场地位
- **LED芯片**: 全球市场份额约30%，中国市场份额约45%
- **化合物半导体**: 中国领先的IDM模式化合物半导体企业
- **碳化硅**: 国内碳化硅衬底和器件主要供应商之一
- **Mini LED**: 苹果等国际大客户供应商

#### 竞争优势

1. **周期性波动**: LED行业周期性明显，价格波动大
2. **技术迭代**: 面临新技术替代风险
3. **国际竞争**: 与科锐、英飞凌等国际巨头竞争
4. **重资产**: 资本开支大，折旧压力大

3. **5G建设**: 射频前端器件需求增长
4. **国产替代**: 半导体国产化政策支持

#### 长期驱动（3-5年）
