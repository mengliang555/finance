# 地平线机器人财务分析报告

## 执行摘要

**公司名称**: 地平线机器人 (Horizon Robotics)
**分析日期**: 2026年3月10日
**分析周期**: 2023-2025年
**投资评级**: 高风险高回报
**目标价格**: 未上市

### 核心结论
地平线机器人作为中国领先的AI芯片和自动驾驶解决方案提供商，在智能汽车和边缘AI计算领域具有重要地位。公司受益于汽车智能化趋势和国产替代需求，但面临技术竞争激烈、研发投入巨大和商业化进程不确定的挑战。作为未上市公司，投资机会主要通过私募股权或未来IPO参与。

## 1. 公司概况

### 1.1 基本信息
- **成立时间**: 2015年7月
- **上市状态**: 未上市（计划IPO中）
- **总部地点**: 北京
- **创始人**: 余凯（前百度深度学习研究院负责人）
- **员工人数**: 约2,000人（2024年）

### 1.2 主营业务
1. **自动驾驶芯片**: Journey系列自动驾驶芯片
2. **边缘AI芯片**: Sunrise系列边缘AI芯片
3. **自动驾驶解决方案**: 全栈自动驾驶解决方案
4. **开发工具链**: 天工开物开发平台

### 1.3 市场地位
- **自动驾驶芯片**: 中国市场份额领先的国产芯片厂商
- **边缘AI芯片**: 在安防、机器人等领域有重要应用
- **技术实力**: 中国AI芯片领域的技术领先者之一
- **客户基础**: 与多家主流车企建立合作关系
