## 使用说明

### 查看版本历史
```bash
# 查看所有文件的提交历史
git log --oneline

# 查看特定文件的更改历史
git log --oneline -- xiaomi_financial_analysis.md

# 查看具体某次提交的更改
git show <commit-hash> -- comparative_analysis_report.md
```

### 恢复历史版本
```bash
# 恢复到特定版本
git checkout <commit-hash> -- xiaomi_financial_analysis.md

# 比较版本差异
git diff <commit1> <commit2> -- meituan_financial_analysis.md
```

### 更新流程
1. 编辑主文件（如 `xiaomi_financial_analysis.md`）
2. 提交更改：`git add . && git commit -m "更新说明"`
3. 推送到远程：`git push origin main`
4. 查看更新：`git log --oneline`
